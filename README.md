# Site sobre a Grécia

Este projeto é um site desenvolvido como atividade acadêmica, com o objetivo de apresentar informações sobre a Grécia de forma organizada, visual e responsiva utilizando Bootstrap.

---

## Sobre o projeto

O site apresenta conteúdos sobre a Grécia:

* Informações gerais do país
* Cultura
* Comidas típicas
* Pontos turísticos

---

## Estrutura do site

O projeto possui 5 páginas principais:

* `index.html` → Página inicial
* `info.html` → Informações gerais
* `cultura.html` → Cultura do país
* `comida.html` → Gastronomia
* `turismo.html` → Pontos turísticos

---

## Tecnologias utilizadas

* HTML5
* CSS3
* Bootstrap 5

---

## Responsividade

O site foi desenvolvido utilizando Bootstrap, garantindo adaptação para:

* Celulares
* Computadores

---

## Funcionalidades

* Navbar responsiva
* Carousel de imagens em todas as páginas
* Layout padronizado
* Footer fixo ao final da página
* Uso de tabelas, listas e imagens
